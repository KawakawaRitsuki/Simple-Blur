# simple-blur.css

## Usage

> .blur-<1-20>px

```HTML
<head>
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/kawakawaritsuki/simple-blur.css@latest/dist/simple-blur.css">
</head>
<div class="blur-5px" style="background: url(<your image>) center; background-size: cover;">
  Text
</div>
```

## Build

```console
$ yarn // or $ npm i
$ npx gulp
```
