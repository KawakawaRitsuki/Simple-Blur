# simple-blur.css

## Usage

### Get Started
```
$ git clone https://github.com/KawakawaRitsuki/simple-blur.css.git
$ cd simple-blur.css/build
$ cp simple-blur.css /path/to/where
```

> .blur-<1-20>px
```
<head>
  ...
  <link rel="stylesheet" href="/simple-blur.css">
</head>
...
<div class="blur-5px">
  Text
</div>
```
## Build

```
$ yarn // or $ npm i
$ npx gulp
```
